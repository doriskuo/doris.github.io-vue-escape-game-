# 🎮 嚇不到我啦！Vue 小遊戲

一款使用 **Vue 3** 開發的互動小遊戲，融合動畫、音效與隨機元素，讓玩家挑戰自己的膽量！  
膽小者慎入，膽大者無妨！😎

---

## 🚀 遊戲特色

- **章節式劇情**：開場動畫 → 第一章 → 第二章 → 結局
- **隨機驚嚇元素**：失敗後會隨機出現不同圖片與音效
- **互動玩法**：快速點擊滑鼠拉滿血條，才能逃出洞窟
- **搞怪 vs. 驚嚇**：既有惡趣味的恐怖音效，也有趣味搞笑的圖片
- **高重玩性**：每次失敗都會有不同驚嚇反饋

---

## 🛠️ 使用技術

- **Vue 3**：最新版本 Vue 框架
- **Vue Router**：多頁面導覽（首頁遊戲、關於頁介紹）
- **組件化設計**：Opening、Game、ChapterOne、ChapterTwo
- **CSS 動畫**：角色動畫、血條動畫、恐怖特效
- **音效觸發**：根據互動狀態切換音效（成功、失敗、追逐等）
- **遊戲邏輯**：
  - Vue `ref` 管理血量與計時
  - `transition` 標籤實現平滑場景切換
  - 動態載入隨機圖片與音效

---

## 🔍 開發挑戰與解法

- **轉場不順問題**  
  - 一開始不同章節切換不流暢  
  - **解法**：使用 Vue `<transition>`，讓畫面淡入淡出，自然銜接

- **隨機圖片無法即時渲染問題**  
  - 嘗試用 `ref` 動態更換 `src`，圖片卻不更新  
  - **解法**：把邏輯拆到另一個 function，利用 `requestAnimationFrame` 強制 DOM 重新渲染，圖片才會更新
